# localize-fixture

An interface class for PrestaShop to generate RTL stylesheets

## Usage
```php
$RTLGenerator::generate($directory);
```

* ``$directory `` : path of admin or front theme

### requirement
``
php >= 5.4
``

``
PrestaShop >= 1.7.2
``
